cardboard-sample
================

[VR Toolkit](https://developers.google.com/cardboard/get-started) ライブラリを用いた、Google Cardboard アプリのサンプル。

Google が公開している [サンプルアプリ](https://github.com/googlesamples/cardboard/)
をベースに、Eclipseプロジェクト化して、日本語対応したものです。
